Signal Protocol

新手 + 老手 + 多设备 含 Rust 骨架 零依赖 · self-contained

Signal Protocol 是当今主流 E2E 加密通信协议（Signal、WhatsApp、Matrix、iMessage 等）的核心算法族—— Double Ratchet 棘轮加密、X3DH 异步首次握手、Sesame 多设备会话管理。 官方规范严谨但门槛较高；本仓库提供多套独立讲法，按读者目标挑路径。

所有图谱都是 self-contained HTML + SVG，无依赖、无构建，浏览器双击即看； 浅色主题（cream + cherry red），适合长时间阅读。
